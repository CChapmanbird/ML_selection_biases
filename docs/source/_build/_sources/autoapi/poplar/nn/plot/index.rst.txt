.. py:function:: loss_plot(train_losses: list, test_losses: list, filename=None)

   :type train_losses: list
   :param test_losses: List of testing data losses
   :type test_losses: list
   :param filename: Output string for the loss curve figure, by default None (in which case, the figure is returned).
   :type filename: str, optional

   :returns: The loss curve matplotlib figure.
